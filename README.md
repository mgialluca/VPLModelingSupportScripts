# VPLModelingSupportScripts

This is a repo by Megan Gialluca with the purpose of connecting her local dev environment (laptop, Trappist) to her hyak Klone enviornment.
Scripts contained in this repo are meant to facilitate the use of Atmos/SMART/LBLABC, individually or in combinations. This repo is not currently in a state to be used easily by others, but is public for the sake of transparency.
